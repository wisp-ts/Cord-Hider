# 🕶️ Cord Hider (MCBE UI Pack)

A lightweight Minecraft Bedrock resource pack that completely hides player coordinates using a custom UI override.

Clean, simple, and effective — no scripts, no lag, just gone.

## 🚀 Features

- ❌ Fully hides coordinates
- 🎯 Works in all worlds
- ⚡ Zero performance impact
- 🧩 Pure UI-based solution
- 🪶 Lightweight (no scripts required)
- 🔒 No gameplay interference
